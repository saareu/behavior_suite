                                                                                                                       
 Usage: sleap-nn predict [OPTIONS]                                                                                     
                                                                                                                       
 Run inference on videos or labels files.                                                                              
 Single unified inference entry point.                                                                                 
                                                                                                                       
+- Options -----------------------------------------------------------------------------------------------------------+
| *  --data_path                              -i  TEXT                      Path to data to predict on. A local       |
|                                                                           Labels (.slp) file or video, or a remote  |
|                                                                           URL (http(s)://, s3://, gs://, gcs://,    |
|                                                                           az://, abfs://) to a .slp or video that   |
|                                                                           sleap-io can load.                        |
|                                                                           [required]                                |
|    --headers                                    TEXT                      JSON object of HTTP headers for a remote  |
|                                                                           --data_path URL, e.g. '{"Authorization":  |

|    --stream-mode,--stream_mode                  TEXT                      Remote read strategy for a --data_path    |

|                                                                           'auto'/'stream'/'download'). Ignored for  |
|                                                                           local inputs.                             |

|                                                                           iff the input was itself an embedded      |
|                                                                           .pkg.slp). Only applies to .slp output.   |

|    urce_videos                                                            references to the original source video   |

|                                                                           representation.                           |
